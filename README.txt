The uploaded files are MetaMazeArduino.ino and MetaMaze.zip
MetaMazeArduino.ino can be run by using the required components on an Arduino Board and uploading hte code on the same.
MetaMaze.zip contains an exe file ArduinoUnityTest.exe.

Certain features have been added to allow the user to use the same WITHOUT an arduino.

1) Press 'z' at the title screen to start the game.
2) When hitting a wall, press 'q' to skip the hardware challenge provided.
3) Press "Alt+F4" to exit the game.
